How To Set Up Your Binance API Keys

Summary

In your Binance account, you can create public and private keys which allow remote connections via the Binance API. CTB uses those keys to authenticate with Binance. This guide explains how to set them.

What Do You Need?

To complete this guide, you will need a verified Binance account.

Steps

  1. Log in to your Binance account.

  2. Click on API Management on your user menu.

  3. Enter a name in the Label API key field, for example, Crypto-Bot, and click Create API. Binance will confirm that it is you and then will redirect you to the API List page.

  4. Click on Edit Restrictions.

  5. Check the Enable Spot & Margin Trading checkbox.

  6. Click on Save.

  7. Copy your API key and your secret key, paste them in the appropriate fields in your user settings under Binance API Keys. Click on Save changes.
