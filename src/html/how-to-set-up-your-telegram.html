How To Set Up Your Telegram

Summary

With Telegram, you can control your CTB from anywhere at any time. This guide shows you how to configure it and how to share your access token and user id with us.

What Do You Need?

To complete this guide, you will need a Telegram account.

Steps

  1. Open your Telegram and go to BotFather. Make sure you choose the verified BotFather, which looks as follows:

  2. Press START and then send the message: /newbot.

  3. BotFather will ask for the public name of the bot. Enter a name, for example, My CTB Trial.

  4. BotFather will then ask for a username for the bot. It must end in bot and can't have spaces in it. Enter a username, for example, MyCTBTrialBot.

  5. BotFather will create the bot and send you a congratulations message containing the HTTP API access token and the link to your newly created bot.

  6. Copy the HTTP API access token and paste it in the Access Token field in your user settings under Telegram Configuration.

  7. Follow the link to your new bot from the congratulations message. The link should look like: t.me/MyCTBbot. Once in your new bot, press START.

  8. Now, go to userinfobot and press START.

  9. Copy the your Id and paste it in the User Id field in your user settings under Telegram Configuration. Click on Save changes.
